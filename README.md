## Recipie-MVVM-Project

A modern, high-performance Flutter application built with **Clean Architecture** and the **MVVM (Model-View-ViewModel)** pattern. This project showcases advanced UI techniques like **Glassmorphism**, smooth state management with **Provider**, and robust networking using **Dio**.

---

### 🚀 Key Features

* **Glassmorphic UI**: High-end visual design using `BackdropFilter` and gradient overlays for a modern look.
* **MVVM Architecture**: Strict separation of concerns between data, logic, and UI.
* **Global State Management**: Centralized logic for recipe fetching and "Favorite" status using `Provider`.
* **Networking**: Efficient API handling and data parsing with the `Dio` package.
* **Responsive Design**: A smooth, card-based layout optimized for mobile devices.

---

### 🛠️ Built With

* **Flutter** - UI Framework
* **Provider** - State Management
* **Dio** - HTTP Client for networking
* **Dart** - Programming Language
